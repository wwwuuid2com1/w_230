<h3>源码简介：</h3>
<p>搭建环境：PHP 5.4 以上 推荐7.0<p>
<p>搭建必备：服务器/主机 域名

搭建教程：
1.源码传到空间 解压
2.解析个域名到空间的IP或CNAME
3.访问域名/install安装
4.安装完访问域名/admin 配置后台信息
5.支付接口：支持支付宝官方pc扫码、手机网站支付、支付宝当面付；支持QQ官方扫码支付、手机支付；支持微信pc扫码、jsapi公众号支付、微信h5；官方码支付等等<p>
